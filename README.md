# Laravel Design Progress

Progress bar and circle component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/progress
```

## Usage

### Livewire Component

```blade
<livewire:ld-progress />
```

### Blade Component

```blade
<x-ld-progress />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-progress-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-progress-views
```
